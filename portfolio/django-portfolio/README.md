# django-portfolio

This is a Django portfolio project designed to showcase personal projects and skills. 

## Features

- User-friendly interface to display portfolio items.
- Responsive design for optimal viewing on various devices.
- Easy to extend and customize with additional features.

## Installation

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/django-portfolio.git
   ```
2. Navigate into the project directory:
   ```
   cd django-portfolio
   ```
3. Create a virtual environment:
   ```
   python -m venv venv
   ```
4. Activate the virtual environment:
   - On Windows:
     ```
     venv\Scripts\activate
     ```
   - On macOS/Linux:
     ```
     source venv/bin/activate
     ```
5. Install the required packages:
   ```
   pip install -r requirements.txt
   ```

## Usage

To run the development server, use the following command:
```
python manage.py runserver
```
Then open your browser and go to `http://127.0.0.1:8000/`.
